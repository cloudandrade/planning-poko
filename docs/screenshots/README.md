# Capturas de tela (portfólio)

Esta pasta contém **mockups de alta fidelidade** das principais telas do **Planning Poko**, alinhados à identidade visual do app (tema escuro + roxo), para uso em portfólio e no `README` principal.

Você pode substituir qualquer arquivo por **screenshots reais** do navegador com o mesmo nome:

| Arquivo | Tela |
|---------|------|
| `01-home.png` | Entrada: criar sala / entrar com código |
| `02-lista-votacoes.png` | Sala: lista de votações (ordem de criação) |
| `03-votacao-ativa.png` | Votação ativa: escolha de cards + painel de votos |
| `04-resultados.png` | Resultados revelados + estimativa final |

**Dica:** use janela 1280×720 ou 1440×900 e capture só o conteúdo da página (sem barra do SO) para manter consistência.
